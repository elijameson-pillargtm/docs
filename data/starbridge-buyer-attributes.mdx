# Starbridge Buyer Attributes

Starbridge enriches PILLAR accounts with 120+ buyer attributes covering identity, geography, budgets, procurement, innovation readiness, demographics, and education-specific metrics. All fields are stored directly on the `accounts` table.

## Identity and Classification

| Attribute | Field | Type | Description |
|-----------|-------|------|-------------|
| Buyer ID | `starbridge_buyer_id` | VARCHAR(50) | Unique Starbridge identifier |
| Common Name | `common_name` | VARCHAR(300) | Short/common organization name |
| Parent Name | `parent_name` | VARCHAR(300) | Parent organization |
| Parent Buyer ID | `parent_buyer_id` | VARCHAR(50) | Parent's Starbridge ID |
| Buyer Type | `buyer_type` | VARCHAR(100) | Buyer type classification |
| District Type | `district_type` | ENUM | K-12 District, Charter, Higher Ed, Municipality, State Agency |
| Classifications | `classifications` | JSONB | Organization classifications array |
