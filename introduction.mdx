# PILLAR Data Reference

PILLAR is an AI-native Go-To-Market Operating System that connects CRM data, district intelligence, and revenue economics into a unified scoring and signal engine.

This reference documents every field, table, scoring rule, and signal type in the PILLAR platform.

## Data Sources

| Source | Type | Sync | Fields |
|--------|------|------|--------|
| Salesforce | CRM | Every 5 min | Accounts, Contacts, Opportunities, Leads |
| HubSpot | CRM | Every 5 min | Accounts, Contacts, Deals, Tickets |
| Google Workspace | Email/Calendar | Per-user, every 5 min | Activities (email metadata, calendar events) |
| Slack | Notifications | Real-time | Signal routing, weekly digest |
| Starbridge.ai | District Intelligence | On-demand + webhook | 120+ buyer attributes, RFPs, contracts, job changes |
| PILLAR Engine | Computed | Every 5 min | Health scores, risk scores, signals, plays |
